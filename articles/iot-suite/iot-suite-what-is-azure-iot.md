## <a name="azure-iot-suite"></a>Azure IoT Suite

De Microsoft Azure IoT Suite is een enterprise-grade-oplossing waarmee u kunt snel aan de slag door een reeks extensible vooraf geconfigureerde oplossingen voor veelvoorkomende IoT-scenario's, zoals [externe controle] [ lnk-preconfigured-solutions] en [Bekijk onderhoud][lnk-predictive-maintenance]. Deze oplossingen zijn implementaties van de IoT oplossingsarchitectuur die in dit artikel beschreven.

De vooraf geconfigureerde oplossingen zijn voltooid, werktijd, end-to-end-oplossingen die bevatten gesimuleerd apparaten om u aan de slag, vooraf geconfigureerde Azure services zoals [Azure IoT Hub][], [Azure gebeurtenis Hubs][] [Azure Stream analyses][], [Azure Machine Learning][], en [Azure opslag][]en oplossing specifieke beheerconsoles. De vooraf geconfigureerde oplossingen bevatten beproefde, productie gereed code die u kunt aanpassen en uitbreiden implementatie van uw eigen specifieke IoT-scenario's.
